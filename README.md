**SpaceAudit: Visual Disk Usage Analyzer with PyQt6**

SpaceAudit is a desktop application built with PyQt6 that scans folders recursively and visualizes disk usage in a hierarchical tree view. It helps users identify space-hogging directories and files, and export detailed reports to Excel with built-in charting.

**Features**

Recursive folder scanning with size and last modified date

Tree-based visualization with percentage breakdown

Export to Excel with formatted hierarchy and pie chart

Dark theme support via qdarkstyle

Built using PyQt6 and xlsxwriter

**Requirements**

Python 3.6+

PyQt6

qdarkstyle

xlsxwriter

**Getting Started**
pip install PyQt6 qdarkstyle xlsxwriter
python spaceAudit.py
